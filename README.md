# LOAD_CELL_PCB_MOUNT

A custom PCB designed to securely mount a load cell sensor using M2 screws.
This board was designed using **KiCad** and prepared for **PCB fabrication**.

---

## Features

* Mechanical PCB designed for **load cell mounting**
* **M2 screw holes (2.2 mm)** for proper fit
* **Non-plated mounting holes (NPTH)** to prevent metal contact with screws
* Designed with **KiCad 9**
* Ready-to-manufacture **Gerber files included**

## Manufacturing

The PCB can be fabricated using any PCB manufacturer such as:

* JLCPCB
* PCBWay
* Seeed Fusion

Simply upload the **Gerber files** from the `GERBER_FILE.zip`.
